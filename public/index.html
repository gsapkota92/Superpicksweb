Trading Signals API

Use these endpoints to integrate Super Picks with automated trading systems. Base URL: https://superpickstrading.com

GET /api/signals
Trading signals optimized for bot consumption. Returns action (STRONG_BUY, BUY, HOLD), confidence score, and current price.
{ "timestamp": "2025-01-15T15:30:00.000Z", "market_open": true, "count": 5, "signals": [ { "symbol": "NVDA", "action": "STRONG_BUY", "signal": "Strong Buy", "score": 7.8, "price": 148.50, "change_pct": 2.3, "confidence": 0.89, "updated": "2025-01-15T15:25:00" } ] }
GET /api/picks
Full technical pick details with all indicator breakdowns (momentum, MACD, SMA, stochastic, volume, CCI, Williams %R).
{ "picks": [ { "symbol": "NVDA", "name": "NVIDIA Corp", "composite_score": 7.8, "overall_signal": "Strong Buy", "price": 148.50, "daily_change": 2.3, "momentum_score": 0.9, "momentum_label": "Bullish", "macd_score": 0.8, "macd_label": "Bullish Crossover", "sma_score": 1.0, "sma_label": "Above SMA", "stoch_score": 0.6, "stoch_label": "Bullish", "volume_score": 0.7, "volume_label": "High Volume", "cci_score": 0.5, "cci_label": "Bullish", "willr_score": 0.3, "willr_label": "Neutral", "updated_at": "2025-01-15T15:25:00" } ] }
GET /api/alpha
Smart money alpha signals from unusual activity detection. Includes conviction scores, tier rankings, and source breakdown (dark pool, insider, options flow).

Fundamental analysis scores with financial metrics. Each stock receives a 1-10 score based on valuation, profitability, growth, and financial health.

Market sentiment score with index performance, sector breakdown, and breadth data. Score 0-100 (Fear to Greed).

GET /api/stats
Overall performance statistics -- win rate, average return, best/worst trades, and recent scan logs.
GET /api/history?status=closed&limit=50
Historical trades with entry/exit prices and returns. Filter by status: active, closed, or all.

Robinhood Integration Example
Python example for automated trading using the signals endpoint:
import requests import robin_stocks.robinhood as rh # Fetch Super Picks signals signals = requests.get("https://superpickstrading.com/api/signals").json() # Login to Robinhood (use your own auth) rh.login(username="your_email", password="your_password") for signal in signals["signals"]: if signal["action"] == "STRONG_BUY" and signal["confidence"] > 0.8: # Place a market order rh.orders.order_buy_market( symbol=signal["symbol"], quantity=1, timeInForce="gfd" ) print(f"Bought {signal['symbol']} at ${signal['price']}") rh.logout()
