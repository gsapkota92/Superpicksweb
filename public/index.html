Trading Signals API

Use these endpoints to integrate Super Picks with Robinhood agentic trading or any automated system. Base URL: https://superpickstrading.com

GET /api/signals
Trading signals optimized for bot consumption. Returns action (STRONG_BUY, BUY, HOLD), confidence score, and current price.
{ "timestamp": "2025-01-15T15:30:00.000Z", "market_open": true, "count": 5, "signals": [ { "symbol": "NVDA", "action": "STRONG_BUY", "signal": "Strong Buy", "score": 7.8, "price": 148.50, "change_pct": 2.3, "confidence": 0.89, "updated": "2025-01-15T15:25:00" } ] }
GET /api/picks
Full pick details with all indicator breakdowns (momentum, MACD, SMA, stochastic, volume, CCI, Williams %R).
{ "timestamp": "2025-01-15T15:30:00.000Z", "count": 5, "picks": [ { "symbol": "NVDA", "name": "NVIDIA Corp", "composite_score": 7.8, "overall_signal": "Strong Buy", "price": 148.50, "momentum_score": 0.9, "macd_score": 0.8, "sma_score": 1.0, "..." } ] }
GET /api/stats
Overall performance statistics — win rate, average return, best/worst trades, and recent scan logs.
GET /api/history?status=closed&limit=50
Historical trades with entry/exit prices and returns. Filter by status: active, closed, or all.
POST /api/picks
Push picks from the Sparsh app. Requires X-API-Key header.
// Header: X-API-Key: your-secret-key { "picks": [ { "symbol": "NVDA", "name": "NVIDIA Corp", "compositeScore": 7.8, "overallSignal": "Strong Buy", "price": 148.50, "dailyChange": 2.3, "indicators": { "momentum": { "score": 0.9, "label": "Bullish" }, "macd": { "score": 0.8, "label": "Bullish Crossover" }, "sma": { "score": 1.0, "label": "Above SMA" }, "stochastic": { "score": 0.6, "label": "Bullish" }, "volume": { "score": 0.7, "label": "High Volume" }, "cci": { "score": 0.5, "label": "Bullish" }, "williamsR": { "score": 0.3, "label": "Neutral" } } } ], "scanMeta": { "totalScanned": 130, "scanTimeMs": 45000 } }
Robinhood Integration Example
Python example for Robinhood agentic trading using the signals endpoint:
import requests import robin_stocks.robinhood as rh # Fetch Super Picks signals signals = requests.get("https://superpickstrading.com/api/signals").json() # Login to Robinhood (use your own auth) rh.login(username="your_email", password="your_password") for signal in signals["signals"]: if signal["action"] == "STRONG_BUY" and signal["confidence"] > 0.8: # Place a market order rh.orders.order_buy_market( symbol=signal["symbol"], quantity=1, timeInForce="gfd" ) print(f"Bought {signal['symbol']} at ${signal['price']}") rh.logout()
